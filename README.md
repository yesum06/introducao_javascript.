# Estudo: 🚀 Cadastro de Categorias em JS
Simulador simples de cadastro para blog usando **JavaScript** para consolidar conceitos fundamentais de lógica de programação. O script simula o sistema de cadastro de categorias de um blog.

### 🧪 Conceitos Aplicados:
* `prompt()` e `alert()`
* Conversão para `Number`
* Laço de repetição `for`
* Condicionais `if/else`
