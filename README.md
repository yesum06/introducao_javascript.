
# Estudo: 📚::Cadastro de Categorias em JS
Simulador simples de cadastro para blog usando **JavaScript** para consolidar conceitos fundamentais de lógica e proporcionar o primeiro contato com a linguagem.

# 🚀::Prática

### 📁:: Cadastro de Categorias
 Pequeno simulador de cadastro para blog usando Js puro.

 ### 🔓:: Login 
  Tela de login com alternância de tema claro/escuro com `classList.toggle`.

## 🪓:: Conceitos Aplicados
* `prompt()` e `alert()`;
* Conversão para `Number`;
* Laço de repetição `for`;
* Condicionais `if/else`;
* Eventos com `addEventListener`;
* Manipulação com DOM.
